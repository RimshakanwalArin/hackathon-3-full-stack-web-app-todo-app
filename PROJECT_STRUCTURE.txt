1. specs/ (CRITICAL)
   └─ Complete specification for all features
   └─ REST API endpoints
   └─ Database schema
   └─ UI component designs
   └─ User flows

2. .spec-kit/
   └─ Configuration for Spec-Kit tooling
   └─ Project metadata

3. frontend/ (TO BE POPULATED)
   └─ Next.js frontend application
   └─ React components
   └─ Page layouts
   └─ API routes

4. .specify/
   └─ Spec-Driven Development (SDD) framework
   └─ Templates for specs, plans, tasks
   └─ Helper scripts

5. history/ (TO BE CREATED)
   └─ Prompt History Records (PHRs) for every user prompt
   └─ Architecture Decision Records (ADRs)
   └─ Feature development history
